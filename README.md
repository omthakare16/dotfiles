 ## **Environment**

```
- OS                   : ubuntu 22.04
- Window manager       : AwesomeWM
- Terminal             : kitty
- Application Launcher : rofi
- Discord Theme        : BetterDiscord (LilyPichu)
- Spotify Theme        : Spicetify (Comfy)
- Wallpaper Engine     : Variety Feh

```
